About iPOJO Nature for Eclipse

iPOJO Nature is an Eclipse plug-in from OW2 Chameleon project providing a project nature and a builder for Java and PDE projects. It is based on the iPOJO manipulator, so it is compatible with metadata.xml and Java annotations component descriptions.

It automates the manipulation of class files by iPOJO after their compilation by the Java plug-in. This way, the developer can use an OSGi Run/Debug Configuration without using an external tool nor a Maven plug-in to run or test an iPOJO components.
