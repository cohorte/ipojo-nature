About iPOJO Builder

iPOJO Builder is an Eclipse plug-in providing a project nature and a builder for Java projects. It is based on the iPOJO manipulator, so it is compatible with metadata.xml and Java annotations component descriptions.

It automates the pojoization of class files after their compilation by the Java plug-in. This way, the user use an OSGi Run/Debug Configuration without using an external tool or a Maven plug-in to use an iPOJO component.
